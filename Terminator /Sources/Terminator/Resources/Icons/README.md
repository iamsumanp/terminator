Place provider logo files in this folder using these exact base names:

- openai
- gemini
- anthropic
- docsumo
- local

Recommended:

- PNG, 64x64 or 128x128, transparent background
- simple, high-contrast glyph/logo

Examples:

- openai.png
- gemini.png
- anthropic.png
- docsumo.png
- local.png

The app will automatically use these images in tab icons.
If a file is missing, it falls back to a built-in letter badge.
